🛒 Simple E-Commerce UI
A sleek and responsive e-commerce web interface built with HTML, Tailwind CSS, and Vanilla JavaScript. It features dynamic product rendering, a popular products section, and an interactive shopping cart with hover-based expansion.

✨ Features
🧾 Product listings with images, descriptions, and prices

⭐ Popular items carousel

🛍️ Add-to-cart functionality with cart preview

💅 Beautiful UI powered by Tailwind CSS & Remix Icons

⚡ Responsive design with fluid scrollable layouts

🔧 Technologies Used
HTML5 + TailwindCSS

JavaScript (ES6)

Remix Icon CDN
